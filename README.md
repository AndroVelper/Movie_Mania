**Movie App: A Clean Architecture Android Project**
This project is a foundational Android application that showcases the implementation of Clean Architecture principles. It is designed to help developers understand Android fundamentals while learning how to manage events effectively in a structured and scalable way.

With features like dependency injection, pagination, and responsive UI, this project serves as a great starting point for developers aiming to build robust Android applications.

**Features**
1. **Hilt** for Dependency Injection
Simplifies dependency management by injecting class instances automatically, reducing boilerplate code.
Ensures seamless lifecycle management of dependencies.
2. **Paging** 3 Library
Implements efficient pagination to handle large datasets.
Ensures smooth scrolling and optimizes network and memory usage.
3. **Retrofit** with Logging
Provides a robust HTTP client to interact with APIs.
Integrated logging to monitor API requests and responses for easier debugging.
4. **NavGraph** for Navigation
Manages navigation between multiple fragments/screens effortlessly.
Ensures a consistent and user-friendly flow throughout the app.
5. **Dynamic and Responsive UI**
Offers a dynamic and responsive design that adapts to various device sizes.
Includes a search feature for better user interaction.
6. **Splash Screen API**
Implements a modern splash screen for a smooth and professional app launch experience.
Technologies Used
**Kotlin:** For app development.
**Hilt:** For dependency injection.
**Paging 3:** For handling paginated data.
**Retrofit:** For API calls with logging capabilities.
**Jetpack Navigation**: For navigation between fragments.
**Splash Screen API:** For a clean and efficient splash screen.

Why This Project?
This project is designed to:

Teach Android Clean Architecture: Learn how to separate concerns for better maintainability and testability.
Demonstrate Modern Android Development (MAD) Practices: Leverage the latest Jetpack libraries and APIs.
Provide a Scalable Codebase: Build a foundation for larger and more complex projects.
